1.HOD login
2.STAFF login

The developers of this website and their qualifiation can be obtained by clicking on About Dev button on the top bar on the screen i.e., at top right corner.

Hod_Login

This page is used for logging.To access further,a valid email and password has to be entered.

Staff Login(I-IV)

This page is used for Staff logging.

REGISTER(Hod password change)

Register is used for creating new id for accessing database.

ADD STUDENT DETAILS

To create student record,click on add details button.

Enter valid details of a student and click on create student record.

After verification of details,click on continue to next page.

Upload photo of a student.To enter academic details of an year,click on that year.

Enter all the values like marks,SGPA,CGPA,student credits,attendence for both semesters.

EDIT STUDENT DETAILS

To edit details of a student,enter their registered roll no. and edit it.

SHOW STUDENT DETAILS

To Display the details of particular student this page is used.student roll number is used to search.

ADD SUBJECTS

This is used for adding subjects for every semester.

Since subjects change every year,they have to be updated for every batch.

CREDITS SEARCH

Used for finding list of students from a year with minimum credits.

For obtaining list who has minimum number of credits,enter year and credit value.This list is diaplayed.

LOGOUT

To logout from the page,click on logout button.

After logout,it returns to the main page of the website.
